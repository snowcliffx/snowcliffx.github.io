Glove.new(player, glove, power, speed, ability)
> Creates new information for a glove.

- **player:** string.
- **glove:** string.
- **power:** number.
- **speed:** number.
- **ability:** string.

```lua
local newGlove = Glove.new("SnowCliffx", "Default", 25, 50, "Blast")<
print(newGlove)
```
